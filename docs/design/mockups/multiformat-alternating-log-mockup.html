Alternating audio & print — one shared spine

The intentional multi-format read: move straight forward, but hand off between formats each sitting — audio ch 1–4, then print ch 5–7, then back to audio. Every session is new ground; they just arrive in different units. Because both editions pin to the same fraction-of-book spine, the chunks tile end-to-end — no gap, no overlap — and the completion bar climbs like any linear read.

2 · Two rulers, one spine. Audio time and print pages are just different rulers on the same book — so an audio chunk and a print chunk sit flush end-to-end

Two rulers, one spine

Every point on the book has an audio time and a page number — two labels for the same spot. That's why alternating never leaves a seam.

Print p.1p.220p.440
The 50% mark is audio 3:35 and print p.220 — the same place. Switch format there and you continue, you don't repeat.
So in the log: the five sittings tile 0→20→35→50→60→70% with no gaps — the frontier only ever moves forward, no matter which ruler you logged in.
The through-line: alternating formats is not a special mode — it's ordinary forward reading where the unit changes between sittings.
